# SQM-Reader
Python SQM Reader for Unihedron SQM-LU or compatible.

### Requirements
* python3
* tkinter
* pyserial

Run program using ``python sqm_reader.py`` or ``python3 sqm_reader.py``. Or, try complited files (not python required!) for Windows or Linux.
